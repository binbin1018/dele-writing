# DELE B2 写作训练

DELE B2 西班牙语写作考试练习工具，支持 AI 批改与满分范文。

## 功能

- **Tarea 1**：书信/邮件写作（150-180词）
- **Tarea 2**：议论文或叙述/描述/评论（150-180词）
- **AI 批改**：支持 DeepSeek 和 ChatGPT (OpenAI GPT-4o)
- **满分范文**：每次批改附带 B2 水平示范作文 + 亮点语法标注

## 使用

访问 [https://binbinliang1018.github.io/dele-writing/](https://binbinliang1018.github.io/dele-writing/)
